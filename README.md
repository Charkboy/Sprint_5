Проект содержит автоматизированные тесты для сайта Stellar Burgers (https://stellarburgers.education-services.ru).

Все тесты независимы и используют фикстуру driver, которая открывает и закрывает браузер для каждого теста.

Локаторы вынесены в отдельный модуль для удобства поддержки

Python 3.14.4

pytest

Selenium WebDriver

ChromeDriver
